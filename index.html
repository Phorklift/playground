Playground

Welcome to Phorklift configuration playground! Phorklift is an HTTP server/proxy daemon with clear, powerful and dynamic configuration. This playground let you to try the configuration online!

If you are here for the first time, it's better to begin with some examples. Select one at the Examples selector, Upload it, Send request to test, and checkout the response. Then try to modify them. Do not forget to Upload before testing.

If you want to modify more or write your own configuration, you may need to read some documents, including configuration tutorial, configuration reference and Lua API reference. You will find that a complete Phorklift configuration always begins with Listen() to define listen address. While each example here is just a Lua table. The reason is that this playground is powered by Phorklift's dynamic-Path configuration. All you need to provide is just a Lua table as Path()'s definition, and it's created dynamicly and run in the same Phorklift process with the playground itself, which is similar to service-discovery. You can find out how this works from its configuration, which is also a typical example of Phorklift's dynamic configuration. If you want to try a complete configuration, you have to run Phorklift yourself.

Generally speaking, the dynamic configurations are always provided by trusted source. You are trusted now. Do not be evil! However there are still some interesting challenges, such as exploring my filesystem or buiding a nested playground.

Feel free to file a Github issue for any question or idea.

Enjoy it!
